# Method Area
- 메소드 영역은 JVM의 메모리 영역인 [[Runtime Data Area(Java)|Runtime Data Area]]의 구성 요소 중 하나로, JVM에서 읽어들인 [[클래스]]와 [[인터페이스]]에 대한 런타임 상수 풀, 메서드와 필드, Static 변수, 메서드 바이트 코드 등을 보관하는 곳이다. 
- Class Area, Static Area 등으로도 불린다.
