# Java에서 DataType(Java)
- [[Java]]에서 [[DataType]]은 변수가 가질 수 있는 데이터의 종류를 정의한다.
- [[Java]]에는 자료형을 다음과 같이 분류한다.
	- [[Primitive Type(java)]]
	- [[Reference Data Type(java)]]
- [[Java]]는 정적으로 데이터 형이 지정된다.
	- 즉, [[Variable(java)|자바의 모든 변수]]는 사용하기 위해서 먼저 데이터 형을 선언해야 한다.
