# Heap 
- 힙은 [[C 언어]]나 [[Java]] 같은 프로그래밍 환경에서 원시 자료형이 아닌 보다 큰 크기의 데이터를 담고자 **동적으로 할당**하는 메모리 공간을 지칭한다.
- 일반적으로 다른 메모리 공간인 [[Call Stack(Memory)]]와 대비되는 공간으로서 존재한다.
	- 그러나 각 언어마다 Heap의 역할이나 구조는 다를 수 있다.
