# Data Access Layer란?
- Data Access(=Persistence) 계층은 데이터 베이스 또는 원격 서비스에서 영구 데이터를 관리하는 역할을 하는 [[layered architecture|layer]]이다.
	- DAO 계층
	- Database / Message Queue / 외부 API와의 통신 등 처리
	- 데이터베이스 또는 원격 서비스에서 영구 데이터를 관리하는 방법을 분류하는 데이터 접근 계층
