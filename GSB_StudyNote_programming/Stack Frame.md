# Stack Frame란?
- 함수 호출과 관련된 실행 컨텍스트를 저장하며(나타내며), 호출된 함수의 [[로컬 변수]], [[매개 변수]], 반환 주소 등을 저장하는 단위.
- 즉, [[Call Stack(Memory)|call stack]]의 저장 단위라고 볼 수 있다.
- 스택 프레임이라는 개념은, ([[Call Stack(Memory)|call stack]]과 마찬가지로) CS 영역 전체에서 일반적으로 사용되는 개념이다.
