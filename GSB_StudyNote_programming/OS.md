# OS란?
- 운영체제는 하드웨어 자원을 관리하고, 응용 프로그램과 하드웨어 사이를 중재하는 인터페이스, 그리고 사용자와 응용 프로그램 간의 인터페이스를 제공하는 프로그램을 말한다.
